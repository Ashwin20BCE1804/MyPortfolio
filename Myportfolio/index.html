Automatic Answer Evaluation Using AI

The AI algorithms used in the project employ natural language processing (NLP) techniques to analyze and understand the semantic meaning of both student responses and the model answer. Python Flask serves as the backbone of the web application, handling data flow and interaction between users and the AI evaluation module hosted on the server. Firebase's real-time database capabilities ensure efficient data storage and retrieval, enabling instant feedback for students and facilitating continuous improvement in their learning journey. Overall, this project addresses the need for efficient and objective evaluation of descriptive answers using the power of Artificial Intelligence, benefiting both educators and students in the learning process.

Admin Dashbaord

Completed the construction of a remarkable React Admin Dashboard, named Shoppy Web Application. This comprehensive dashboard comprises one central dashboard, three dynamic pages, and four feature-rich apps, all backed by seven fully functional and interactive charts for data visualization. The application utilizes powerful tools such as Syncfusion, React hooks, refs, and React context APIs to ensure a visually appealing and responsive user interface, efficient state management, and optimized performance. With this project, I demonstrate my proficiency in developing sophisticated React applications, offering an exceptional user experience and innovative features.

Github profile finder

This project enables seamless searching of GitHub profiles, providing users with essential user information, repositories, and employability status with utmost ease. This project leverages the power of React states and hooks capabilities, enabling dynamic data management and a smooth user experience. By integrating the dynamic GitHub API, users can effortlessly search for GitHub profiles and access relevant details in real-time. The application's intuitive interface and advanced features make it a valuable tool for developers and recruiters alike, streamlining the process of evaluating potential candidates and discovering talented individuals in the GitHub community.

Blog App

Created a blog app using HTML, CSS, PHP, and MySQL involves developing a user-friendly frontend with HTML and CSS, styling it to provide an appealing interface. The PHP backend handles server-side processing, facilitating CRUD operations to manage blog posts efficiently. MySQL is utilized as the database to store blog posts and associated data. By implementing user authentication (optional), registered users can manage their posts securely. The blog app offers a seamless experience, allowing users to create, read, update, and delete blog posts while accessing a well-organized and visually appealing platform.
